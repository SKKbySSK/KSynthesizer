# KSynthesizer
4Eのデザイン実験で用いるシンセサイザーライブラリ

## 構成
- KSynthesizer
    - シンセサイザーのコアライブラリ
    - クロスプラットフォーム対応
    - .Net Standard 2.1, .Net Core 3.0
- TestTool.Windows
    - Windows向けのデバッグツール
    - 波形表示やオシレータの調整が可能
